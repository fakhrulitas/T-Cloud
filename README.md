# T-Cloud App

Aplikasi cloud storage sederhana untuk Android berbasis bot Telegram.

## Spesifikasi & Fitur
* **Minimal Android:** 9.0 (API Level 28).
* **Fitur:** Manajemen file (gambar, video, audio) dan integrasi Telegram.
* **Izin:** Akses internet, media, dan lokasi.

## Cara Pakai
1. Clone repository ini.
2. Buka di Android Studio.
3. Build dan jalankan di perangkat API 28 ke atas.

## Download
Silakan cek bagian **Releases** untuk mengunduh file APK terbaru.
